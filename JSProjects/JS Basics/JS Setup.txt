To Create a new project, Open Visual Studio Code editor 
	1. Open folder created for the project 
	2. beside the folder there will be a +(Folder) icon, hit icon to and a file.
	3. Create a new file and name it index.html
	4. CTRL + B will open the file in a full window.
	5. Type doc and the TAB and this will populate the file with bare bone code.
	6. Type JS code between the script tags and Save file. (Auto Save)
	7. Open a Chrome tab and type CTRL +O and this will open dialog to open html file.
	8. Select the file and open. 
	
Linking HTML to js file
	1. Start with index.HTML from previous instructions
	2. CTRL + N create a new file
	3. Name and save the file. (filename.js)
	4. Use js file name in HTML file 
		Example:  <script src="home.js"></script>
	
	5. You are able to call the HTML tag id to make changes within the js file. 
		Example: \
		[HTML FILE]
		<body>
			<h1 id="testHeader"></h1>
			<p id="someText"></p>
			<script src="home.js"></script>
		</body>
		
		[js File]
		// Changing Data in HTML file using tag id
		document.getElementById('testHeader').innerHTML = 'Test program!!!';
		document.getElementById('someText').innerHTML = 'Hello World';

	6. You are able to concatenate element id too
		Example:
		var age = prompt('What is your age');
		document.getElementById('someText').innerHTML = 'Your age is ' + age;
